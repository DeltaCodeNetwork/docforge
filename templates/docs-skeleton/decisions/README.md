# Decisions (ADRs)

Immutable Architecture Decision Records (MADR) — one decision per file, numbered `NNNN-kebab-title.md`. They capture context, options, the decision, and consequences. **Never edit a decided ADR** — supersede it with a new one and mark the old `superseded by ADR-NNNN`. Create via `/docforge:adr`; template in `_templates/adr-template.md`.
